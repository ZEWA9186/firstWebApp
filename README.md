# firstWebApp

Персональный проект для разработки и тестирования архитектурных решений на стеке Java / Spring Boot.

## Технологический стек:
* **Backend:** Java (LTS), Spring Boot (Security, Data JPA)
* **Database:** PostgreSQL(в бущем будут добавлены новые)
* **Tools:** MapStruct, Lombok(будут обновляться по мере использования)

## Особенности реализации:
* Многослойная архитектура с четким разделением ответственности.
* Использование DTO для изоляции внутреннего состояния системы.
* Централизованная валидация и обработка исключений.

Проект используется как база для внедрения различных функциональных модулей: от систем авторизации до комплексной бизнес-логики.
